2.IO多路复用？select/poll/epoll的区别？
select 
    本质上是设置和检查存放的fd标志位的状态进行判断是否需要下一步处理的
poll
    poll本质上和select是一致的，不过poll没有大小限制，因为poll是通过链表保存的
epoll
    epoll维护了fd和流的映射关系，当读写完成完成时，自动执行回调
